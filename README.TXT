How to Start the game and Instructions:

1. Compile all files given in the folder( there are 5 .java files)
2. run the HW10Driver class and input the according numbers
3. Press a button to begin the game
4. You will win when there are no non-mine tiles left OR you click on a mine
5. You will be asked if you want to restart after a win/lose.(Clicking yes will restart, clicking either no/cancel will 
   put you back into the original game.
   **You might be given an option pane more than once the first time the game is played through** - Just click yes everytime
